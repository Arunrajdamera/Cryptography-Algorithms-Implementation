# Cryptography Algorithms Implementation

This is a simple Python project that demonstrates basic cryptography algorithms:

- AES encryption and decryption
- RSA encryption and decryption
- SHA-256 hashing

## Requirements

- Python
- pycryptodome

Install the required library:

```bash
pip install -r requirements.txt
```

## Run the Project

```bash
python crypto_project.py
```

Enter a message when prompted. The program will show:

- AES encrypted output
- AES decrypted output
- RSA decrypted output
- SHA-256 hash
